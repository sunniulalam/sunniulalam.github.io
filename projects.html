Library Management System
A Library Management System made using JavaSwing for the GUI and MySQL as the database. The functionality includes the ability to search for books in the database using ISBN, Author name, and/or book title, Checking In/ Out books, Manage Borrowers, and Manage Fines.

QuicPic
QuicPic is an Instagram web-clone that uses React and Firebase. The website allows users to post photos or videos on the feed. It also allows users to comment on others' posts. The website checks to ensure that new user creation does not violate database requirements.
https://quicpic-react.web.app/
Comex
Comex is an ongoing project to allow UTD Comets to exchange foreign currencies or cryptocurrencies using the balance on their bursar accounts. It is also meant for traveling students, whether it be going abroad or being an international student, to exchange currencies for their financial benefit. It uses API data to ensure that exchange rates are up to date.
https://github.com/jaredgrady3/jaredgrady3.github.io
